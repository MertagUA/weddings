# Весільне запрошення · В'ячеслав & Дарина 💍

Статичний односторінковий сайт-запрошення (HTML/CSS/JS, без збірки).
Зроблений за сценарієм із ТЗ (`Запрошення на весілля.pdf`): конверт із
печаткою → лист → календар → локація → таймінг → дрес-код → відлік → фінал.
Адаптивний (мобільна + десктоп), з анімаціями та персоналізацією під гостя.

## Екрани (відповідають ТЗ)

| Екран | Розділ                | Що показує                                     |
|-------|-----------------------|------------------------------------------------|
| 1     | Конверт               | Печатка «В & Д», імена, дата · клік відкриває   |
| 2     | Лист                  | «Дорогі рідні та друзі!» + текст запрошення    |
| 3     | Календар              | Вересень 2026, 12-те число у сердечку          |
| 4     | Локація               | «Театр Палас», адреса, фото, кнопка на мапі    |
| 5     | Таймінг               | 13:00 → 22:00, п'ять пунктів програми          |
| 7     | Дрес-код              | Святковий стиль + палітра кольорів             |
| 8     | Відлік                | «Ми скажемо "Так" через…» — лічильник          |
| 9     | Фінал                 | «Святкуйте з нами!», фото, імена               |

## Кольори (з ТЗ)

`#ffe5e3` ніжно-рожевий · `#ffffff` білий · `#6f7d5f` зелений — задані в
`styles.css` як CSS-змінні `--pink`, `--white`, `--green`.

## Фото

Лежать у `images/`:
- `image1.jpeg` — пара (фінальний екран)
- `image3.jpeg` — інтер'єр закладу (локація)
- `image2.jpeg` — пара на природі (запасне, поки не використане)

Щоб поміняти фото — просто замініть файли або відредагуйте теги `<img>`
у `index.html`.

## Персоналізація за посиланням (?id=)

Відкрийте сайт із параметром `?id=` — і конверт та звертання зміняться:

```
.../?id=1   → «Запрошення для мами й тата» · «Дорогі мамо й тату!»
.../?id=2   → «Запрошення для бабусі й дідуся»
.../?id=3   → «Запрошення для Олени» · «Дорога Олено!»
```

Гостей редагуйте у файлі **`guests.js`**. Без `id` (або з невідомим `id`)
показується нейтральне «Запрошення для вас» / «Дорогі рідні та друзі!».

## Що варто перевірити/замінити

- `guests.js` — ваш список гостей
- `index.html` — посилання на Google Maps (зараз пошук за адресою закладу)
- `script.js` → `WEDDING_DATE` — дата й час для відліку (зараз 12.09.2026 13:00)

## Локальний запуск

```bash
npx serve .
# або
python3 -m http.server 8000
```

## Деплой на Vercel

Збірка не потрібна — статика.

**Через сайт:** `git push` → на [vercel.com](https://vercel.com) **Add New → Project** →
імпорт репозиторію → Framework **Other**, Build Command порожній, Output `./` → **Deploy**.

**Через CLI:**
```bash
npm i -g vercel
vercel --prod
```
